## 链表

​	**1）反转链表：（三指针）+递归=每k个反转【25】**

```
    ListNode curr = head;//指向当前要反转的
    ListNode pre = NULL;//指向已经反转的，注意开始要为NULL
        while(curr!=NULL&&n-->0){//每n个反转
            ListNode tem = curr.next;//tem指向下一个
            curr.next=pre;
            pre = curr;
            curr=tem;
        }
        return pre;
    }
```
